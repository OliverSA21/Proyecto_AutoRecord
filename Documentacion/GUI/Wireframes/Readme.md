# Wireframes 
Los wireframes representan la estructura básica de cada pantalla de AutoRecord sin estilos visuales finales. Su función es definir la lógica, jerarquía, distribución de elementos y navegación antes de aplicar color, tipografía o diseño gráfico.

En esta etapa se validan los flujos de usuario, se identifican mejoras en la experiencia y se asegura que cada pantalla cumpla con su propósito funcional.
