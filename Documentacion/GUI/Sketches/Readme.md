# Sketches 
Los sketches son los primeros bocetos rápidos del diseño de AutoRecord. Se utilizan para explorar ideas, definir estructuras iniciales y visualizar la distribución general de cada pantalla antes de pasar a wireframes o mockups.

Esta etapa permite iterar de forma ágil, probar diferentes acomodos y decidir qué elementos deben ir en cada vista de la aplicación móvil y de escritorio.

¿Qué incluyen los sketches de AutoRecord?

<ul>
    <li>Bocetos iniciales de las pantallas principales</li>
    <li>Exploración de diferentes diseños para el menú, mapa y reportes</li>
    <li>Ideas de distribución de elementos en interfaz (inputs, botones, tarjetas)</li>
    <li>Estructura base del flujo del usuario</li> 
    <li>Primeras propuestas para:</li>
    <ul> 
        <li>Pantalla de inicio de sesión</li> 
        <li>Menú principal</li>
        <li>Ubicación de unidades</li> 
        <li>Listado de unidades y conductores</li> 
        <li>Reportes de operación</li> 
    </ul> 
</ul>
