# Mockups 
Los mockups son representaciones visuales de alta fidelidad que muestran cómo se verá la interfaz final de AutoRecord, una aplicación diseñada para gestionar y optimizar rutas, unidades y procesos operativos dentro del transporte público.

En esta parte del proyecto se definen los colores, estilos, iconografía y diseño general que se implementará tanto en la aplicación móvil como en la versión de escritorio.

¿Qué incluyen los mockups de AutoRecord?

<ul>
    <li>Diseño visual de pantallas para la app móvil del checador</li>
    <li>Diseño visual de pantallas para la app de escritorio del administrador</li>
    <li>Componentes visuales como botones, menús, tarjetas, formularios e iconos</li>
    <li>Identidad gráfica del sistema (paleta de colores, estilo y distribución)</li>
    <li>Propuestas de interfaz para las principales funciones:</li>
    <ul>
        <li>Registro de unidades</li>
        <li>Ubicación de rutas</li>
        <li>Reportes operativos</li>
        <li>Perfil de usuario</li>
        <li>Gestión de unidades</li>
    </ul>
</ul>
